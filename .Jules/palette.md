## 2026-04-16 - SVG CSS Tooltip Keyboard Accessibility\n**Learning:** SVG elements with tooltips generated via CSS lacked keyboard accessibility, meaning users navigating with keyboards or screen readers couldn't access tooltip content. Tooltips that rely solely on `:hover` are inherently inaccessible.\n**Action:** When injecting CSS tooltips, always add `tabindex="0"` and an `aria-label` to the target element. Update the CSS to trigger visibility on `:focus` (e.g., `[data-tooltip]:focus + .pathy-tooltip`) alongside `:hover`, and include a clear `:focus-visible` outline.
